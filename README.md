# transcricao-whisper
# 🎙️ Transcrição Automática com Whisper

Este projeto é uma interface simples e poderosa para transcrição de áudio, usando o modelo Whisper da OpenAI com Gradio. Basta enviar um arquivo de áudio (.mp3, .wav, entre outros) e obter a transcrição textual de forma automática.

## ⚙️ Tecnologias
- Whisper (modelo "base")
- Gradio para interface web interativa
- FFmpeg para leitura e decodificação de áudio

## 🚀 Como usar
1. Clone o repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Adicione o caminho do FFmpeg ao ambiente (se necessário)
4. Execute o script Python em seu Jupyter Notebook
5. Faça upload de um áudio e receba sua transcrição

## 🎯 Objetivo
Facilitar o acesso à transcrição automática de áudios para uso em podcasts, entrevistas, vídeos e mais.

## 📌 Observações
- A transcrição é feita localmente, sem envio de dados para servidores externos.
- Você pode trocar o modelo "base" por versões mais rápidas ou mais precisas: "tiny", "small", "medium", "large".
